# Reproducible Workflow

A reproducible workflow makes it possible to rerun an analysis from raw inputs to final outputs.

## Suggested Structure

```text
project/
  data/
  code/
  output/
  docs/
  README.md
```

## Basic Rules

- Keep raw data unchanged.
- Put generated outputs in a separate folder.
- Use scripts for repeatable work.
- Commit meaningful changes with Git.
