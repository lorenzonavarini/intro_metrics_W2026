## Course description

The course provides an introduction to the most common statistical methods that are used in modern economics. In particular, the course will introduce students to the concept of linear regression analysis discussing how to set up and estimate linear regression models and conduct statistical inference with them. In doing so, the course will provide an understanding of causal inference and, more generally, guidelines for interpreting empirical regression results and the corresponding hypothesis tests. The course will also treat implementation issues related to regression analysis, such as misspecification or omitted variables. Empirical applications in Stata and R illustrate the methods.

## Course Goals

The course emphasizes three dimensions:

1. Econometric reasoning and identification
2. Applied data work and reproducible computation
3. Clear communication of empirical evidence

## Syllabus

### Session 1: Development Environment

- Install and configure the software stack
- Set up Git and GitHub
- Organize a reproducible project folder
- Run scripts from the terminal

### Session 2: Data, Measurement, and Description

- Load and inspect datasets
- Create derived variables
- Produce summary tables and figures
- Document data-cleaning choices

### Session 3: Regression and Interpretation

- Estimate linear models
- Interpret coefficients
- Work with controls and fixed effects
- Report results cleanly

### Session 4: Causality and Research Design

- Distinguish prediction from causal interpretation
- Work through omitted-variable bias
- Introduce experimental and quasi-experimental designs

### Session 5: Project Workflow

- Structure an empirical project
- Track changes with Git
- Prepare a replication package
- Present empirical findings
